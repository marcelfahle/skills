# Banned Patterns: The Forensic Kill List

Every word, phrase, structure, and pattern in this file is a statistically verified marker of AI-generated text. Presence of ANY of these in output is a failure.

---

## Tier 1: Dead Giveaway Words (Zero Tolerance)

These words appear in AI text at 10x-200x human baseline rates. Using even ONE flags the text.

### The "Smart" Verbs
| Banned | Use Instead |
|--------|------------|
| Delve / Delves into | Dig into, Look at, Examine, Study, Check |
| Leverage (as verb) | Use, Apply, Exploit, Capitalize on |
| Utilize | Use |
| Showcase | Show, Present, Display |
| Underscore | Highlight, Show, Prove |
| Unleash | Release, Unlock, Free up |
| Harness | Use, Tap into, Put to work |
| Foster (a culture/environment) | Build, Grow, Encourage, Create |
| Embark (on a journey) | Start, Begin, Set out |
| Unveil | Show, Reveal, Announce |
| Facilitate | Help, Enable, Make possible |
| Empower | Give, Equip, Enable |
| Navigate (metaphorical) | Handle, Deal with, Work through |
| Streamline | Simplify, Speed up, Cut |
| Optimize | Improve, Fix, Tune |
| Spearhead | Lead, Drive, Run |

## Tier 2: Structural Tells (Pattern Level)

### Banned Transitions & Connectives
| Banned | Use Instead |
|--------|------------|
| Furthermore | Also, Plus, And, (or just start the sentence) |
| Moreover | Besides, On top of that, What's more |
| Additionally | Also, And, Plus, (or just start) |
| In conclusion | (Delete. Just conclude.) |
| Ultimately | (Delete or use "Bottom line:") |
| It is important to note | (Delete entirely. If it's important, it speaks for itself.) |
| It is worth mentioning | (Delete entirely.) |
| It is crucial to recognize | (Delete entirely.) |
| In today's [anything] | (Delete. Start with the actual point.) |
| In the ever-evolving | (Delete. Burn it.) |
| At its core | (Delete.) |
| That being said | But, Still, However (sparingly) |
| Consequently | So, Because of this, That's why |
| Thus | So |
| Notably | (Delete. If it's notable, the reader will notice.) |
| In order to | To |
| With that in mind | (Delete.) |
| As such | So |
| In light of | Given, Because of, After |
| Moving forward | Next, From here, Going ahead |

### Banned Openings
Never start any piece with:
- "In today's fast-paced world..."
- "In the ever-evolving landscape of..."
- "In the realm of..."
- "As we navigate the complexities of..."
- "[Topic] has become increasingly important..."
- "It's no secret that..."
- "Picture this:" (when used as generic hook)
- "Have you ever wondered..."
- "Whether you're a [X] or a [Y]..."
- "In recent years..."

### Banned Closings
Never end any piece with:
- "In conclusion..."
- "Ultimately..."
- "At the end of the day..."
- "The future is bright..."
- "Only time will tell..."
- "...paving the way for a brighter future"
- "...a testament to the resilience of the human spirit"
- "...remains to be seen, but offers hope"
- "The ball is in your court"
- "So what are you waiting for?"

### Banned Sentence Patterns
- The participial opener: "Standing tall, the building..." / "Having completed X, she..."
- The triple parallel: "It's not just X; it's Y, Z, and W"
- The "Not only...but also" construction (once per 2000 words max)
- Nominalization chains: "The implementation of the optimization of the process"
- Passive hedging: "It can be seen that..." / "It is generally considered..."
- The sycophantic opener: "Great question!" / "That's an excellent point!"
- The safety preamble: "Before we dive in, it's important to understand..."

## Tier 3: Punctuation & Formatting Tells

### Em Dashes
**BANNED.** Em dashes (—) are now the single most flagged punctuation mark in AI detection. AI overuses them 3-5x human baseline. Replace with:
- Periods (for separation)
- Commas (for light pauses)
- Colons (for introduction/explanation)
- Parentheses (for asides)
- Semicolons (sparingly, for connected independent clauses)

### Formatting Patterns
- Never use 3 bullet points of equal length (the "AI triad")
- Never use bullet points for more than 20% of total content
- Never use "Header: Explanation" repeated identically for 3+ items
- Avoid numbered lists unless the sequence genuinely matters
- Never bold more than 5% of text
- Never use emoji in professional content unless platform demands it (social media)

## Tier 4: Tonal Tells

### The "HR Voice" — Banned Tonal Patterns
- Relentless optimism: Every paragraph ending on a positive note
- Bothsidesism: Presenting "balance" where a clear position is warranted
- Sycophancy: Agreeing with every premise, praising every question
- False empathy: "I understand how you feel" / "It's understandable that"
- Moral preaching: Unsolicited ethics disclaimers
- Over-qualification: "While this may vary depending on context..."
- The "Disney ending": Wrapping everything in a neat, hopeful bow

### The "Wikipedia Voice" — Banned Epistemic Patterns
- Neutral-to-a-fault: No opinion, no stake, no skin in the game
- Citation simulation: "Research suggests..." / "Studies have shown..."
- Abstract hedging: "Some experts believe..." / "It is generally accepted..."
- The "to be fair" reflex: Giving equal airtime to fringe positions

## Tier 5: Structural Architecture Tells

### The "5-Paragraph Essay Ghost"
- Introduction that previews everything
- 3 body paragraphs of similar length
- Conclusion that summarizes everything
- **Kill it.** Professional writing doesn't follow this template.

### The "Hamburger Paragraph"
- Topic sentence stating the point
- 2-3 supporting sentences
- Concluding sentence restating the point
- **Kill it.** Not every paragraph needs this structure. Let some paragraphs be one sentence. Let others sprawl.

### The "Rule of Three" Addiction
AI defaults to 3 examples, 3 points, 3 reasons. Always.
- Use 1 vivid example instead of 3 generic ones
- Use 2 when that's all you need
- Use 4 or 5 to break the pattern
- Never 3 unless it's genuinely the right number

### Uniform Sentence Length
If sentence lengths cluster between 15-25 words with standard deviation under 7 words, the text reads as synthetic. Target standard deviation above 10.
