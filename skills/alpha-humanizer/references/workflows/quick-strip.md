# Quick Strip Workflow

Emergency 5-minute edit protocol. When the user needs fast results, not a full rewrite.

## When to Use
User needs speed. "Just clean this up quick." Or the text is mostly fine but has obvious AI tells.

## Prerequisites
- Read `references/concepts/banned-patterns.md` (at minimum, the Tier 1 banned words)

## Steps (5 Steps, 1 Minute Each)

### Step 1: Kill the First Paragraph
If it starts with "In today's," "In the ever-evolving," any landscape/realm language, or a generic context-setter, delete it entirely. Start with the second paragraph. If the second paragraph also starts with throat-clearing, start with the third.

### Step 2: Ctrl+F Destroy
Find and replace/delete every Tier 1 banned word:
- Delve → examine, dig into, look at
- Leverage → use
- Utilize → use
- Tapestry → (delete metaphor, be specific)
- Landscape → market, field, industry
- Robust → strong, solid
- Seamless → smooth, easy
- Furthermore/Moreover/Additionally → (delete, or use "And," "Plus," "Also")
- In conclusion → (delete)
- It is important to note → (delete)
- In order to → to

Remove all em dashes. Replace with periods or commas.

### Step 3: Break One Paragraph
Find the longest, densest paragraph. Split one sentence into a standalone one-line paragraph. This single visual break changes the perceived rhythm of the entire piece.

### Step 4: Add One Specific Detail
Find one abstract phrase ("many users," "a significant increase," "the industry") and replace with a specific: a number ("42% of users"), a name ("Instagram's algorithm"), a sensory detail ("the 3 AM email you'll regret").

### Step 5: Fix the Ending
If it ends with "In conclusion," "Ultimately," or a Disney bow ("paving the way for a brighter future"), delete the last sentence. End on the second-to-last sentence. If that one is also generic, rewrite the last line as a direct statement or question.

## Output Format
Deliver the cleaned text directly. No explanation unless asked.

## Quick Verification
After the 5 steps, read the text once. If you can predict the next word in more than 2 sentences, those sentences need one more pass.
